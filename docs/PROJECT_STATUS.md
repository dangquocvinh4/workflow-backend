# PROJECT STATUS - Spa Booking System

## 📌 Tổng quan dự án
Dự án xây dựng hệ thống đặt lịch hẹn Spa (Spa Booking System) bao gồm Frontend (React + Vite) và Backend (Spring Boot + PostgreSQL).

## 🛠 Tech Stack
- **Frontend**: React 19, Vite, Zustand (State Management), Axios, Tailwind CSS v4.
- **Backend**: Java Spring Boot, Spring Data JPA, Spring Security.
- **Database**: PostgreSQL (NeonDB).
- **Tooling**: Antigravity Kit (.agent), NotebookLM (via nlm-mcp-cli).

## 📊 Trạng thái hiện tại (Phase 4 - Testing & Deploy Readiness)

- **UX**: Tích hợp Toast notification, Loading states và Route Guards.
